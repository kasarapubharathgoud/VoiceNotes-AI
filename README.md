# 🎙️ VoiceNotes AI

AI-powered web application that converts text notes into voice-based learning.

---

## 🚀 Features
- 🎤 Voice Input
- 🌍 Multi-language Support 
- 🔊 Text-to-Speech Output
- ⚡ Fast Processing
- 🎛 Adjustable Speed
- 🔗 Murf AI Integration

---

## 💡 Problem
Students find it difficult to read long notes and retain information effectively.

---

## ✅ Solution
VoiceNotes AI converts text into natural speech so users can listen instead of reading, improving learning efficiency.

---

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Web Speech API
